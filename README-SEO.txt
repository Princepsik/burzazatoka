SEO úpravy bez výrazných změn designu:

- nový <title> a meta description pro fráze „burza“ a „starožitnosti“
- canonical URL a robots meta
- Open Graph metadata pro sdílení
- schema.org LocalBusiness JSON-LD
- doplněný úvodní text s klíčovými výrazy
- lepší hierarchie nadpisů H1/H2
- alt texty u obrázků
- lazy loading obrázků
- klikací telefon, email a web
- opraven odkaz na FB.png (v ZIPu byl velkými písmeny)
- jemně vylepšená responzivita bez změny vzhledu
